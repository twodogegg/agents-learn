# Skill 结构

这里解释一个 skill 通常由哪些部分组成。

推荐结构：

```text
my-skill/
  SKILL.md
  scripts/
  templates/
  references/
  assets/
```

常见组成：

- `SKILL.md`：skill 的入口，说明什么时候使用、怎么执行
- `scripts/`：可复用脚本
- `templates/`：输出模板或文件模板
- `references/`：较长的参考资料
- `assets/`：图片、示例文件、静态资源

一个好 skill 的重点不是文件多，而是能让 agent 稳定完成同一类任务。
