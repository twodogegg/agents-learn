# Skill Metadata

这里放 skill metadata 的参考说明。

可以记录：

```yaml
name: code-review
description: Run a comprehensive code review.
category: workflow
tags:
  - coding
  - review
  - quality
risk: low
status: active
```

建议字段：

- `name`：skill 名称
- `description`：触发描述
- `category`：主分类
- `tags`：辅助标签
- `risk`：风险等级
- `status`：active / deprecated / experimental
