# Skill 分类总览

几十个 skills 不建议直接平铺。更好的方式是按任务场景分类。

推荐分类：

| 分类 | 解决的问题 |
| --- | --- |
| Workflow Skills | 组织 agent 的工作方式 |
| Coding Skills | 软件开发、测试、重构、发布 |
| Research Skills | 资料获取、信息分析、source checking |
| Writing Skills | 写作、翻译、改写、排版 |
| Visual Skills | 图片、图表、设计、视觉检查 |
| Browser / Automation Skills | 浏览器、桌面、自动化操作 |
| Office / Document Skills | PDF、Word、PPT、Excel |
| Knowledge Base Skills | 笔记、wiki、知识库 |
| Integration Skills | 第三方平台和业务系统连接 |
| Dangerous / Experimental Skills | 高风险、实验性、真实账号操作 |

一个 skill 最好只有一个主分类，但可以有多个 tags。
