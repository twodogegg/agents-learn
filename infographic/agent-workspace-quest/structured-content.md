# Structured Content

## Title

Agent Workspace 5-Step Quest

## Learning Objective

Show how to build a reusable AI agent workspace from core ideas to repeatable workflows.

## Sections

### 1. Prompt

- Key concept: define task intent
- Content: 写清目标、上下文、约束
- Visual element: scroll / magic note icon
- Label: 设定任务

### 2. Agent

- Key concept: assign role and behavior
- Content: 选择角色、instructions、memory
- Visual element: pixel hero character
- Label: 召唤 Agent

### 3. Skill

- Key concept: package reusable ability
- Content: 把稳定流程写成 skill
- Visual element: skill card / spell book
- Label: 解锁 Skill

### 4. MCP

- Key concept: connect external tools
- Content: 接入 tools、resources、prompts
- Visual element: portal / cable node
- Label: 连接 MCP

### 5. Workspace

- Key concept: organize files and workflow
- Content: 固化目录、配置、复盘
- Visual element: castle / base camp
- Label: 建成 Workspace

## Design Instructions

- Horizontal game level progression
- Retro 8-bit pixel game style
- Bright blue sky, grassy platforms, mountains, coins, hearts, quest UI
- Similar energy to a SNES educational game screen
- Keep all labels readable
